# Data Sources: Reusable-Inventory-RL

## Recommended Dataset
**Kaggle: High-Dimensional Supply Chain Inventory Dataset**
- Link: [High-Dimensional Supply Chain Inventory](https://www.kaggle.com/datasets/thyagomartins/high-dimensional-supply-chain-inventory-dataset)
- Description: SKU-level data, sales, inventory levels, and lead times. Perfect for training RL agents for inventory control.

## Instructions
1. Download the supply chain inventory dataset from Kaggle.
2. Extract the features relevant to replenishment and stock levels.
3. Place in `data/`.
